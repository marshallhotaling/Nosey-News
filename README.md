## Installation 

1. Fork this FE repo
   [Nosey News Hole FE repo](https://github.com/marshallhotaling/Nosey-News)
2. `cd` into cloned repo 
3. Run `npm install`
4. Run `npm start`
5. Enter `control + c` in your terminal to stop running the React app at any time
